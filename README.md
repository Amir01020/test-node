# iCorp Interview Task

Тестовое задание для позиции Node.js разработчика в iCorp.

## Описание

Скрипт для взаимодействия с тестовым API, демонстрирующий навыки HTTP-коммуникации и обработки данных.

## Алгоритм работы

1. Отправка POST-запроса на `https://test.icorp.uz/interview.php` с JSON payload
2. Получение первой части кода в ответе
3. Получение второй части кода через webhook
4. Конкатенация обеих частей
5. Отправка GET-запроса с объединенным кодом
6. Получение исходного сообщения

## Установка
```bash
npm install
```

## Запуск

1. В первом терминале запустите скрипт:
```bash
node index.js
```

2. Во втором терминале запустите ngrok:
```bash
ngrok http 3000
```

3. Скопируйте HTTPS URL из ngrok и вставьте в первый терминал

## Зависимости

- axios
